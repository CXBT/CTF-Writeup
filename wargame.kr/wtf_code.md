# WTF CODE

## 문제

```
This is another programming language.

Can you read this source code?
```

## 풀이

문제에 들어가면 아무것도 보이지 않는데 소스코드라고 한다. 주어진 파일 `source_code.ws`를 보면 띄어쓰기 외엔 다른 코드가 보이지 않는다.

주어진 파일은 `Whitespace`이라는 프로그래밍 언어로 쓰인 코드로 안보이는 문자로만 코드를 구성한 언어이다. `Whitespace` 인터프리터를 인터넷에서 어떻게 찾아서 주어진 파일을 넣어 실행하면 플래그를 찾을 수 있다.
